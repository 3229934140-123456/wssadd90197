## 1. 产品概述

面向医美集团老板和区域运营负责人的移动端风险监控App，聚焦储值资金风险而非日常收银。帮助管理者在手机上随时掌握全集团储值余额、消耗趋势、异常充值行为，并能即时审批超额赠金、特殊退款等高风险操作，实现在外巡店时快速判断和决策。

- 目标用户：医美集团老板、区域运营负责人、门店管理者
- 核心价值：将储值资金风险从"事后发现"转变为"实时预警+移动审批"，降低退费压力和资金沉淀风险

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 集团老板 | 查看全集团数据、审批所有类型、管理重点观察名单 |
| 区域运营负责人 | 查看所辖区域数据、审批区域内的操作、区域门店观察 |
| 门店管理者 | 查看本店数据、发起审批申请、查看本店会员 |

### 2.2 功能模块

1. **今日概览**：全集团储值余额、今日新增储值、今日消耗、待退款金额、红黄绿风险概览
2. **门店排行**：按门店比较储值增长与消耗偏离度、重点观察名单管理、门店风险评级
3. **异常会员**：充值异常提醒、顾问集中操作提醒、频繁拆单提醒、会员风险详情
4. **审批消息**：待审批列表、审批操作（批准/驳回）、原因选择、语音备注、审批历史

|----------|----------|----------|

## 3. 核心流程

**主流程**：管理者打开App → 查看今日概览掌握全局 → 发现风险信号 → 进入门店排行/异常会员排查 → 对需审批事项进行审批 → 将重点关注门店加入观察名单

**审批流程**：收到审批推送 → 查看审批详情（含会员风险标签） → 选择批准或驳回 → 必选原因 + 可选语音备注 → 审批完成

## 4. 用户界面设计

### 4.1 设计风格

- **主题色调**：深色背景（#0F1419）配金融级蓝绿主色（#00D4AA），辅以风险红（#FF4757）、预警黄（#FFA502）、安全绿（#2ED573）
- **设计语言**：移动优先的Dashboard风格，数据密度高但不拥挤，卡片式布局
- **字体**：数字使用 DIN Alternate 等金融级等宽字体突出数据，正文使用思源黑体/Noto Sans SC
- **布局**：底部Tab导航（4个主模块），顶部状态栏，内容区可滚动
- **图标风格**：线性图标（Lucide），与数据卡片搭配
- **动效**：数据变化时数字滚动动画，风险信号脉冲动画，卡片进入渐显

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 今日概览 | 资金总览卡片 | 深色卡片、大字数字、同比环比小字、渐变底色区分 |
| 今日概览 | 风险信号条 | 红黄绿圆点+简短文字，脉冲动画提示高风险 |
| 今日概览 | 消耗趋势图 | 折线图，深色背景，蓝绿色线条，区域填充 |
| 门店排行 | 排行列表 | 列表项含排名、店名、偏离度柱状、风险标签 |
| 门店排行 | 重点观察 | 顶部固定区域，金色边框卡片，优先展示 |
| 异常会员 | 异常提醒 | 三种提醒类型用不同颜色区分，右侧箭头可展开 |
| 异常会员 | 会员详情 | 时间线竖向布局，项目列表卡片式 |
| 审批消息 | 审批列表 | 左侧类型图标+颜色条，右侧待审批角标 |
| 审批消息 | 审批操作 | 底部弹出操作面板，原因多选，语音按钮带波形动画 |

### 4.3 响应式设计

- 移动优先设计，最大宽度480px居中
- 支持触摸操作，按钮最小44px触控区
- 下拉刷新、上拉加载更多
- 横屏时图表自适应宽度

### 4.4 3D场景指引

- 不适用，本项目为2D数据展示型App
